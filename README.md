# Arch_Setup

1. Sync packages
```sh
pacman -Syu
```

2. Install git
```sh
pacman -S git
```

3. Clone repo
```sh
git clone https://github.com/Plunne/Arch_Setup.git
```

4. Run script
```sh
./Arch_Setup/arch_install.sh
```

5. When you'll be stopped in chroot, run `Arch_Setup/arch_chroot.sh`.

6. Setup your desktop/wm

7. Enjoy!
